# OpenQinling-cpu

#### 介绍
秦岭系列cpu

#### CPU架构/参数
详情见各cpu版本的帮助文档


#### 使用教程

1.  选择cpu版本
2.  进入core源码包
3.  根据帮助文档复制core软核源码到工程中，并根据模板实例化cpu模块即可
4.  在工具包中找到对应cpu版本的编译器
5.  根据自身需求使用该cpu构建soc
